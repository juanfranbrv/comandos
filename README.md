# Proyecto de Automatización con Python

Este proyecto contiene scripts para automatizar la creación de entornos de desarrollo en Python, incluyendo la configuración de entornos virtuales, la instalación de dependencias, la inicialización de repositorios Git y la creación de proyectos con Streamlit.

## Estructura del Proyecto

- **`streamlit-go.py`**: Script para crear proyectos con Streamlit.
- **`python-go.py`**: Script para crear proyectos básicos en Python.
- **`requirements.txt`**: Archivo con las dependencias necesarias para ejecutar los scripts.
- **`.gitignore`**: Archivo para excluir carpetas y archivos innecesarios del control de versiones.

## Requisitos Previos

- Python 3.7 o superior.
- [Git](https://git-scm.com/).
- [GitHub CLI](https://cli.github.com/).
- [Visual Studio Code](https://code.visualstudio.com/).
- Módulo `colorama` instalado (se incluye en `requirements.txt`).

## Instalación

1. Clona este repositorio o descarga los archivos.
